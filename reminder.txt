
variables to pass to lili view:

currentParent
currentItems
items
limboItems

routes that lili view posts to:
/setAsCurrent
/createItem
/updateItemText
/toLimbo
/deleteItem
/fromLimbo

'/'
  se existir cookie de usuario:
    redireciona para view da lili do usuário
  se não:
    redireciona para página de conectar ou experimentar

'/view da lili'

'/connect or try'
  redireciona para connect ou try com <a>

'/try'
  cria novo rootid e currentid
  cria usuário sem connectid
  cria cookie de usuário com  _id, rootid e currentid
  redireciona para '/'

'/connect'
  passa pela validação na rede social e retorna connectid
  se existir usuário com o connectid:
    carrega o cookie com _id, rootid e currentid do usuário existente
    redireciona para '/'
  se não:
    se existir um cookie de usuário:
        criar connectid no usuário con _id do cookie.
        redireciona para '/'
    se não:
      criar novo rootid e currentid, cria usuário com connectid.
      redireciona para '/'
